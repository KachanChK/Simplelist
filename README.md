# Simplelist

Simplelist - Keep it simple. Minimalista e focado no essencial.

![Simplelist Preview](./public/screenshot.png)

## Funcionalidades

- **To-do List:** Lista de tarefas com persistência local.

Sinta-se a vontade para contribuir para o projeto com novas funcionalidades e melhorias.

## Tecnologias

- React (com TypeScript)
- Vite
- Tailwind CSS

## Como rodar o projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/KachanChK/Simplelist.git
   ```

2. **Entre na pasta:**
   ```bash
   cd simplelist
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Rode o projeto localmente:**
   ```bash
   npm run dev
   ```
